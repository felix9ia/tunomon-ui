# Tunomon-UI

使用umi-library构建




- 使用typescript编写


## 使用

`npm install tunomon-ui`

## 调试

切换到此组件目录下，然后执行

`npm link`

切换到项目根目录，然后执行

`npm link tunomon-ui`

防止冲突

删除node_modules下除了软连接生成的`tunomon-ui`之外的带有`tunomon-ui`的其他目录
